# 🌱 ReLife – AI Resource Rescue Network

## About the Project
ReLife is an AI-powered donation platform that helps reduce waste by connecting donors with NGOs, schools, orphanages, and people in need.

## Problem Statement
Millions of usable items such as books, clothes, electronics, and furniture are discarded every year while many people lack access to these essential resources.

## Solution
ReLife provides a smart platform where users can donate unused items. AI categorizes donations, recommends suitable NGOs, and simplifies the donation process.

## Features
- User Login and Registration
- Donate Items
- AI Item Categorization
- NGO Dashboard
- Donation Tracking
- Responsive Design

## Technologies Used
- Lovable AI
- React
- Tailwind CSS
- AI-powered Interface
